# Theme Guide

### Visual Identity

#### Color Palette

* **Primary Green:** #10B981 (trust/money)
* **Dark Slate:** #1F2937 (text/backgrounds)
* **Light Gray:** #F9FAFB (cards/UI)
* **Accent Orange:** #F97316 (alerts/CTAs)
* **Success Green:** #22C55E (confirmations)

#### Themes (In-App)

Users can choose per-chat:

* **Light Mode** (default): White bg, dark text
* **Dark Mode:** Dark slate bg, white text
* **Coming Soon:** Ocean blue, sunset orange, mint green

### UI Components

**Buttons**

* Primary: Green (#10B981)
* Secondary: Outlined
* Danger: Red for delete actions

**Cards**
